# sunnyJS

一个简洁的 Javascript 库。

A simple Javascript library.

## 目标

- 轻量级，可定制，低耦合，无依赖
- 保持同 jQuery、Zepto 一致的 API，方便迁移。
- 同时支持移动平台和浏览器。

## 特别的一些东西

纪念某些事情、带来的触动；

sunny [adj.]  和煦的; 照到阳光的; 快活的，性情开朗亲切的;

希望整个项目能为你带来一缕阳光.
